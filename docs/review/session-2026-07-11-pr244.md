### 5 pattern áp dụng đồng nhất

1. **Preload kho từ SP** trong `boot()` / `render()` → truyền mảng rút gọn xuống view.
2. **Alpine local JS search** — `filtered = filterXxx(search)` trên state Alpine.
3. **Bỏ `wire:model` khi gõ** — dùng `x-model="search"`; sync Livewire qua `$wire.set()` chỉ khi chọn item / Enter / Tab.
4. **Normalize bỏ dấu** — `normalizeText()`: NFD + bỏ combining marks + `đ→d` + lowercase + gom space.
5. **Commit exact code trước, fallback dòng đầu filtered** — `commitSearch()`.
