# AR - Stored Procedures Documentation

Tài liệu chi tiết cho các Stored Procedures module AR.

## Danh mục SPs

| Stored Procedure | Chức năng | Tài liệu |
|------------------|-----------|----------|
| `asARGetDMKH` | Lấy danh mục khách hàng theo điều kiện | [asARGetDMKH.md](stored-procedures/asARGetDMKH.md) |

## Quy ước đặt tên

- **asAR\***: Business logic AR (AR = Accounts Receivable)
- **asARGet\***: Lấy dữ liệu (SELECT)
- **asARInsert\***: Thêm mới (INSERT)
- **asARUpdate\***: Cập nhật (UPDATE)
- **asARDelete\***: Xóa (DELETE)
- **asARPost\***: Post dữ liệu sang GL
